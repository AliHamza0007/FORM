# Local Storage Form with HTML, JavaScript, and Bootstrap

This is a simple web application that allows users to input data into a form, store that data locally using browser's local storage, and then display the stored data on the webpage. The application is built using HTML, JavaScript, and Bootstrap.


## Features

- Create and submit a form with user data.
- Store submitted data locally using browser's local storage.
- Display stored data on the webpage.
- Clear stored data.

## Usage

1. **Opening the Application**

   - Clone this repository to your local machine or download the ZIP file and extract it.

   ```bash
   git clone https://github.com/yourusername/your-local-storage-form.git
   ```

2. **Open the Project**

   - Navigate to the project directory and open the `index.html` file in your web browser.

   ```bash
   cd your-local-storage-form
   ```

3. **Using the Form**

   - Fill out the form fields with the required information.
   - Click the "Submit" button to store the data locally.
   - The stored data will be displayed on the webpage below the form.

4. **Clearing Stored Data**

   - To clear all stored data, click the "Clear Data" button.

## Installation

There is no need for installation, as this is a client-side web application. Simply open the `index.html` file in your web browser.

## Development

If you want to customize or further develop this project, here are some guidelines:

- Modify the `index.html` file to change the form structure and layout.
- The JavaScript logic for handling form submission and local storage is in the `script.js` file. You can modify it to suit your needs.
- You can also enhance the styling by editing the `styles.css` file.
- Test your changes by opening the `index.html` file in your web browser.
